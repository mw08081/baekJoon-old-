## 등차수열
등차수열은 일정한 수만큼, 즉 공차(d)씩 증가하는 수열을 의미한다  

예를 들어 `1, 3, 5, 7, 9 ...`가 있으며 이 수열은 2씩 증가하기때문에 수열의 공차는 2가 된다  

An번째 등차수열의 값은 `An = A1 + (n - 1) * d`이고  
An번째까지의 등차수열의 합은 `Sn = n * (2 * A1 + (n - 1) * d) / 2 = n * (A1 + l) / 2 (단, l = An)`이다
